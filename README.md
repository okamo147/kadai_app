1. アプリの概要

・目的: 日本の人口統計データを可視化し、地域ごとの推移を直感的に把握するためのWebアプリです 。

・公開URL: https://kadaiapp-ebyjqebrnydjgltf27isfa.streamlit.app/

・リポジトリURL: https://github.com/okamo147/kadai_app

2. 使用データについて 

・統計表名: 統計ダッシュボード 人口指標

・データセット名/表番号: 人口（系列コード: 0301010001010010010030301） 

・データ取得日: 2026年1月31日 


・各列の意味: 

　・@time: 調査年（西暦）

　・$: 人口（単位：人）

　・regionCode: 地域コード（都道府県別）

3. 機能とUI 

・サイドバー操作: 都道府県の切り替え、およびグラフの色のカスタマイズが可能

・可視化: 折れ線グラフ（人口推移）とデータテーブル（詳細数値）の2種類を表示

・動的表示: ユーザーの選択に応じて、グラフとテーブルがリアルタイムに更新される

4. 独自実装：未使用UI部品の利用
公式 API Reference を調査し、以下の部品を実装

１．st.color_picker: 

　・目的: ユーザーが自分好みの色でグラフを表示できるようにするため 

　・箇所: サイドバーの「グラフの線の色を選択」 


２．st.dataframe の高度な表示: 

　・目的: 数値の最大値を強調表示（ハイライト）し、データの傾向を掴みやすくするため 

　・箇所: メイン画面の「詳細データ一覧」 


３．st.download_button: 

　・目的: 表示されているデータをCSV形式でローカルに保存できるようにするため 

　・箇所: サイドバー下部
