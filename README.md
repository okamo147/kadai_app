1. アプリの概要

・目的: 日本の人口統計データを可視化し、地域ごとの推移を直感的に把握するためのWebアプリです 。

・公開URL: (ここにStreamlit CloudのURLを貼る) 

・リポジトリURL: (ここにGitHubのURLを貼る) 

2. 使用データについて (必須記載事項)

・統計表名: 統計ダッシュボード API 

・データセット名/表番号: 人口（系列コード: 0301010001010010010030301） 

・データ取得日: 2026年1月31日 


・各列の意味: 

　・@time: 調査年（西暦）

　・$: 人口（単位：人）

　・regionCode: 地域コード（都道府県別）

3. 機能とUI (3つの柱)

・サイドバー操作: 都道府県の切り替え、およびグラフの色のカスタマイズが可能です 。

・可視化: 折れ線グラフ（人口推移）とデータテーブル（詳細数値）の2種類を表示します 。

・動的表示: ユーザーの選択に応じて、グラフとテーブルがリアルタイムに更新されます 。

4. 独自実装：未使用UI部品の利用 (3つ以上)
公式 API Reference を調査し、以下の部品を実装しました 。

１．st.color_picker: 

　・目的: ユーザーが自分好みの色でグラフを表示できるようにするため 。

　・箇所: サイドバーの「グラフの線の色を選択」 。


２．st.dataframe の高度な表示: 

　・目的: 数値の最大値を強調表示（ハイライト）し、データの傾向を掴みやすくするため 。

　・箇所: メイン画面の「詳細データ一覧」 。


３．st.download_button: 

　・目的: 表示されているデータをCSV形式でローカルに保存できるようにするため 。

　・箇所: サイドバー下部 。
